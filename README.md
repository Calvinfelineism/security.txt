# Calvinfelineism Studios

**Calvinfelineism Studios** is an independent studio passionate about Android app development, web design, and indie game production for Steam.

## Who We Are

We blend Reformed theology, creative worldbuilding, and digital innovation—crafting content that is both doctrinally grounded and deeply engaging. Our core strengths include:

- **Reformed theological writing**
- **Pixel and narrative game design**
- **Sound doctrine blogging**
- **Creative worldbuilding**

## Featured Projects

- **[Calvinfelineism.org](https://www.calvinfelineism.org):** Theological blog with digital catechism and Reformed reflections.
- **AETHERIUM: Shadows of the Past:** Sci-fi theological indie game (in development).
- **CalvinFeline Dev Tools:** Internal web tools for Christian creators and solo developers.

## Our Mission

> To glorify God through creativity, code, and story.

We unite doctrine and digital craftsmanship, striving for projects that edify, inspire, and entertain.
